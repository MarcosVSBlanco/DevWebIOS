Exercício 2

Crie um Array unidimensional com 14 posições contendo os seguintes valores 17, 43, 8, 4, 97, 56, 29, 3, 75, 34, 88, 82, 61 em cada posição irá conter um valor do tipo number.

Utilizando Arrow Function com o método filter crie uma verificação para retornar somente os valores que estão entre o número 20 e 80.
